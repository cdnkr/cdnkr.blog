Tracking a user's location is essential for applications that provide localized experiences, such as setting default locations in maps, weather updates, or delivery services. There are two primary ways to determine a user's location:

1. **IP-based location tracking** - Uses third-party services to estimate a user's location based on their IP address.
2. **Browser-based Geolocation API** - Uses the device's built-in geolocation capabilities to get precise location data (requires user permission).

## 1. IP-Based Location Tracking

IP-based tracking determines a user's approximate location based on their IP address. Since raw IP addresses don't inherently store location data, third-party services like [ipapi.co](https://ipapi.co/) maintain large databases that map IP ranges to geographic locations. These services use data from ISPs, network infrastructure, and public databases to provide location estimates.

### Why Can't You Determine Location from an IP Address Yourself?

- **Lack of Access to ISP Databases**: Internet Service Providers (ISPs) assign IP addresses dynamically, and only they know the exact user-to-IP mapping.
- **Use of VPNs and Proxies**: Many users hide their actual IP using VPNs or proxies, making it difficult to determine their true location.
- **Databases Need Constant Updates**: IP addresses get reassigned frequently, so maintaining an accurate database requires continuous updates.

### Fetching the User's Location via IP

```javascript
const getIpApiData = async () => {
    try {
        const response = await fetch(`https://ipapi.co/json`);
        if (!response.ok) {
            throw new Error(`HTTP error! Status: ${response.status}`);
        }
        const data = await response.json();
        return data;
    } catch (err) {
        console.error("Error fetching IP data:", err);
        return { latitude: 51.5074, longitude: -0.1278 }; // Default to London
    }
};
```

### How It Works
- Calls `ipapi.co/json` to retrieve IP-based location data.
- Returns latitude and longitude if successful.
- Defaults to London if the request fails.

**Limitations**:
- Only provides an approximate location (city level at best).
- Can be inaccurate due to VPNs, mobile networks, or ISP routing.

## 2. Browser-Based Geolocation API

The **Geolocation API** provides precise location data but requires user permission to comply with privacy laws (e.g., GDPR, CCPA). This method directly accesses the device's GPS, Wi-Fi positioning, or cellular triangulation for more accurate results.

### Why Does It Require Permission?

- **Privacy Concerns**: Unlike IP-based tracking, which is passive, this method actively retrieves sensitive location data.
- **Security Risks**: Allowing websites unrestricted access to precise user locations could lead to tracking abuse.
- **User Control**: Browsers ensure that users explicitly allow or deny access for each request.

### Fetching the User's Location via Browser API

```javascript
const getPreciseLocation = () => {
    if (navigator.geolocation) {
        navigator.geolocation.getCurrentPosition(
            (position) => {
                const location = {
                    lat: position.coords.latitude,
                    lng: position.coords.longitude,
                };
                console.log("Precise location:", location);
            },
            (error) => {
                console.error("Error fetching precise location:", error);
            }
        );
    } else {
        console.error("Geolocation is not supported by this browser");
    }
};
```

### How It Works
- Checks if the browser supports `navigator.geolocation`.
- Calls `getCurrentPosition`, prompting the user for permission.
- If granted, retrieves accurate coordinates.

**Limitations**:
- Requires explicit user permission.
- Doesn't work if users deny access.

## 3. Displaying Both Locations with Mapbox

We'll now integrate **Mapbox** into a React app to display the user's IP-based and precise locations.

### Step 1: Install Dependencies

```sh
npm install react-map-gl mapbox-gl
```

### Step 2: Create the Map Component

```javascript
import React, { useState, useEffect } from "react";
import ReactMapGL, { Marker } from "react-map-gl";
import "mapbox-gl/dist/mapbox-gl.css";

const MapComponent = () => {
    const [viewport, setViewport] = useState({
        longitude: -0.1278,
        latitude: 51.5074,
        zoom: 10,
    });
    const [ipLocation, setIpLocation] = useState(null);
    const [preciseLocation, setPreciseLocation] = useState(null);

    useEffect(() => {
        (async () => {
            const ipData = await getIpApiData();
            setIpLocation({ lat: ipData.latitude, lng: ipData.longitude });
        })();
    }, []);

    return (
        <ReactMapGL
            {...viewport}
            width="100%"
            height="500px"
            mapboxAccessToken={process.env.NEXT_PUBLIC_MAPBOX_ACCESS_TOKEN}
            mapStyle="mapbox://styles/mapbox/dark-v11"
            onMove={(evt) => setViewport(evt.viewState)}
        >
            {ipLocation && (
                <Marker longitude={ipLocation.lng} latitude={ipLocation.lat}>
                    <div className="marker ip">IP Location</div>
                </Marker>
            )}
            {preciseLocation && (
                <Marker longitude={preciseLocation.lng} latitude={preciseLocation.lat}>
                    <div className="marker precise">Precise Location</div>
                </Marker>
            )}
        </ReactMapGL>
    );
};

export default MapComponent;
```

## 4. Calculating Distance Between Locations

Using the **Haversine formula**, we calculate the great-circle distance:

```javascript
const calculateDistance = (lat1, lon1, lat2, lon2) => {
    const R = 6371; // Earth's radius in km
    const dLat = (lat2 - lat1) * (Math.PI / 180);
    const dLon = (lon2 - lon1) * (Math.PI / 180);
    const a = Math.sin(dLat / 2) ** 2 +
        Math.cos(lat1 * (Math.PI / 180)) * Math.cos(lat2 * (Math.PI / 180)) *
        Math.sin(dLon / 2) ** 2;
    return R * (2 * Math.atan2(Math.sqrt(a), Math.sqrt(1 - a))); // Distance in km
};
```

import mapboxgl from "mapbox-gl";
import ReactMapGL, { Marker, Source, Layer } from "react-map-gl";

import "mapbox-gl/dist/mapbox-gl.css";

import Button from "../ui/button";

/**
 * Fetches user's location data based on IP address
 * @returns {Promise<Object>} Location data including latitude and longitude
 */
const getIpApiData = async () => {
    try {
        const response = await fetch(`https://ipapi.co/json`);

        if (!response.ok) {
            throw new Error(`HTTP error! Status: ${response.status}`);
        }

        const data = await response.json();
        return data;
    } catch (err) {
        console.error("Error fetching IP data:", err);
        return defaultResponse;
    }
};

/**
 * Default location data to use if IP-based location fails
 * Uses London coordinates as a fallback
 */
const DEFAULT_LOCATION = {
    latitude: 51.5074, // London coordinates as fallback
    longitude: -0.1278,
    country_code: "GB",
    ip: "0.0.0.0"
};

/**
 * Calculates the distance between two geographic points using the Haversine formula
 * @param {number} lat1 - Latitude of first point in degrees
 * @param {number} lon1 - Longitude of first point in degrees
 * @param {number} lat2 - Latitude of second point in degrees
 * @param {number} lon2 - Longitude of second point in degrees
 * @returns {number} Distance in kilometers
 */
const calculateDistance = (lat1, lon1, lat2, lon2) => {
    const R = 6371; // Radius of the earth in km
    const dLat = deg2rad(lat2 - lat1);
    const dLon = deg2rad(lon2 - lon1);
    const a =
        Math.sin(dLat / 2) * Math.sin(dLat / 2) +
        Math.cos(deg2rad(lat1)) * Math.cos(deg2rad(lat2)) *
        Math.sin(dLon / 2) * Math.sin(dLon / 2);
    const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1 - a));
    const distance = R * c; // Distance in km
    return distance;
};

/**
 * Converts degrees to radians
 * @param {number} deg - Angle in degrees
 * @returns {number} Angle in radians
 */
const deg2rad = (deg) => {
    return deg * (Math.PI / 180);
};

/**
 * Calculates the midpoint between two geographic coordinates
 * @param {number} lat1 - Latitude of first point
 * @param {number} lng1 - Longitude of first point
 * @param {number} lat2 - Latitude of second point
 * @param {number} lng2 - Longitude of second point
 * @returns {Object} Midpoint coordinates {lat, lng}
 */
const calculateMidpoint = (lat1, lng1, lat2, lng2) => {
    return {
        lat: (lat1 + lat2) / 2,
        lng: (lng1 + lng2) / 2
    };
};

/**
 * Map component that displays user's IP-based location and precise location
 * Allows users to compare the two locations and see the distance between them
 */
const MapComponent = () => {
    // State for storing location data and UI state
    const [ipLocation, setIpLocation] = useState(null);
    const [preciseLocation, setPreciseLocation] = useState(null);
    const [loading, setLoading] = useState(true);
    const [error, setError] = useState(null);
    const [distance, setDistance] = useState(null);
    const [viewState, setViewState] = useState({
        longitude: DEFAULT_LOCATION.longitude,
        latitude: DEFAULT_LOCATION.latitude,
        zoom: 10
    });
    const mapRef = useRef(null);

    /**
     * Effect hook to fetch the user's IP-based location on component mount
     */
    useEffect(() => {
        const fetchIpLocation = async () => {
            try {
                setLoading(true);
                // Get location data from IP
                const ipData = await getIpApiData();

                const location = {
                    lat: ipData.latitude,
                    lng: ipData.longitude
                };

                setIpLocation(location);
                setViewState({
                    longitude: location.lng,
                    latitude: location.lat,
                    zoom: 10
                });
            } catch (err) {
                console.error("Error fetching IP location:", err);
                setError("Failed to get location from IP");
                // Use default location
                setIpLocation({
                    lat: defaultResponse.latitude,
                    lng: defaultResponse.longitude
                });
            } finally {
                setLoading(false);
            }
        };

        fetchIpLocation();
    }, []);

    /**
     * Gets the user's precise location using the browser's Geolocation API
     * Calculates distance from IP-based location and adjusts map view
     */
    const getPreciseLocation = () => {
        if (navigator.geolocation) {
            setLoading(true);
            navigator.geolocation.getCurrentPosition(
                (position) => {
                    const newLocation = {
                        lat: position.coords.latitude,
                        lng: position.coords.longitude
                    };
                    setPreciseLocation(newLocation);

                    // Calculate distance if both locations are available
                    if (ipLocation) {
                        const dist = calculateDistance(
                            ipLocation.lat, ipLocation.lng,
                            newLocation.lat, newLocation.lng
                        );
                        setDistance(dist);
                    }

                    // Adjust map to show both points
                    if (mapRef.current && ipLocation) {
                        const map = mapRef.current.getMap();

                        // Create bounds that include both points
                        const bounds = new mapboxgl.LngLatBounds();
                        bounds.extend([ipLocation.lng, ipLocation.lat]);
                        bounds.extend([newLocation.lng, newLocation.lat]);

                        // Fit the map to these bounds with padding
                        map.fitBounds(bounds, {
                            padding: { top: 100, bottom: 50, left: 100, right: 100 },
                            duration: 1000
                        });
                    }

                    setLoading(false);
                },
                (err) => {
                    console.error("Geolocation error:", err);
                    setError("Failed to get precise location. Please check your browser permissions.");
                    setLoading(false);
                }
            );
        } else {
            setError("Geolocation is not supported by your browser");
        }
    };

    /**
     * GeoJSON data for the route line between IP and precise locations
     */
    const routeData = {
        type: "Feature",
        properties: {},
        geometry: {
            type: "LineString",
            coordinates: ipLocation && preciseLocation ? [
                [ipLocation.lng, ipLocation.lat],
                [preciseLocation.lng, preciseLocation.lat]
            ] : []
        }
    };

    /**
     * Style configuration for the route line
     */
    const lineLayer = {
        id: "route",
        type: "line",
        paint: {
            "line-color": "#fafafa",
            "line-width": 3,
            "line-dasharray": [2, 1]
        }
    };

    /**
     * Custom marker component with label and pin
     * @param {Object} props - Component props
     * @param {string} props.color - Background color of the pin
     * @param {React.ReactNode} props.children - Content to display in the label
     */
    const CustomMarkerPin = ({ color, children }) => (
        <div className="flex flex-col items-center">
            <div className="text-white text-xs font-mono uppercase bg-black px-2 py-1 mb-1 flex flex-col gap-1">
                {children}
            </div>
            <div
                className="w-5 h-5 rounded-full border-2 border-white cursor-pointer"
                style={{ backgroundColor: color }}
            />
        </div>
    );

    /**
     * Component to display the distance between locations
     * @param {Object} props - Component props
     * @param {number} props.distance - Distance in kilometers
     */
    const DistanceLabel = ({ distance }) => (
        <div className="bg-black px-2 py-1 text-white text-xs font-mono uppercase">
            <p className="font-bold">{distance.toFixed(2)} km</p>
        </div>
    );

    // Mapbox dark style URL
    const mapStyle = "mapbox://styles/mapbox/dark-v11";

    // Show loading state if map data isn't ready
    if (loading && !ipLocation) {
        return <div>Loading map...</div>;
    }

    return (
        <div className="not-prose w-full relative">
            {/* Display any errors that occurred */}
            {error && <div className="text-red-500">{error}</div>}

            {/* Render map once IP location is available */}
            {ipLocation && (
                <ReactMapGL
                    ref={mapRef}
                    mapboxAccessToken={process.env.NEXT_PUBLIC_MAPBOX_ACCESS_TOKEN}
                    {...viewState}
                    onMove={evt => setViewState(evt.viewState)}
                    style={{ width: "100%", height: "500px" }}
                    mapStyle={mapStyle}
                >
                    {/* IP Location Marker */}
                    <Marker
                        longitude={ipLocation.lng}
                        latitude={ipLocation.lat}
                        anchor="bottom"
                    >
                        <CustomMarkerPin color="#000000">
                            <p className="text-xs font-mono uppercase font-bold">IP Location</p>
                            <p className="text-xs font-mono uppercase">
                                lat <span className="text-gray-300">{ipLocation.lat.toFixed(2)}</span> lng <span className="text-gray-300">{ipLocation.lng.toFixed(2)}</span>
                            </p>
                        </CustomMarkerPin>
                    </Marker>

                    {/* Precise Location Marker - only shown after user grants permission */}
                    {preciseLocation && (
                        <Marker
                            longitude={preciseLocation.lng}
                            latitude={preciseLocation.lat}
                            anchor="bottom"
                        >
                            <CustomMarkerPin color="#000000">
                                <p className="text-xs font-mono uppercase font-bold">Precise Location</p>
                                <p className="text-xs font-mono uppercase">
                                    lat <span className="text-gray-300">{preciseLocation.lat.toFixed(2)}</span> lng <span className="text-gray-300">{preciseLocation.lng.toFixed(2)}</span>
                                </p>
                            </CustomMarkerPin>
                        </Marker>
                    )}

                    {/* Distance Label at Midpoint - shows how far apart the locations are */}
                    {ipLocation && preciseLocation && distance && (
                        <Marker
                            longitude={calculateMidpoint(ipLocation.lng, ipLocation.lng, preciseLocation.lng, preciseLocation.lng).lng}
                            latitude={calculateMidpoint(ipLocation.lat, ipLocation.lat, preciseLocation.lat, preciseLocation.lat).lat}
                            anchor="top-right"
                        >
                            <DistanceLabel distance={distance} />
                        </Marker>
                    )}

                    {/* Dashed line connecting the two location points */}
                    {ipLocation && preciseLocation && (
                        <Source id="route-source" type="geojson" data={routeData}>
                            <Layer {...lineLayer} />
                        </Source>
                    )}
                </ReactMapGL>
            )}

            {/* Button to trigger getting the user's precise location */}
            <div className="w-full flex justify-center mt-6">
                <Button
                    onClick={getPreciseLocation}
                    disabled={loading || !ipLocation}
                >
                    {loading ? "Getting location..." : "Show precise location"}
                </Button>
            </div>
        </div>
    );
};

export default MapComponent;
